# Links Page

This is a repo containing an example links page built with Bulma Clean Theme and Jekyll. 

If you want a simple landing page with links to your social media profiles and other important links, then [fork this repo](https://github.com/chrisrhymes/LinksPage/fork) to get started. 

Check out the full documentation for the links page in the [Bulma Clean Theme documentation site](https://www.csrhymes.com/bulma-clean-theme/docs/links-pages/creating-a-links-page/).
